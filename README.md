# Проект: Определение возраста покупателей по фотографии

## 1. Назначение и цель проекта

В прикассовой зоне внедряется система компьютерного зрения для обработки фотографий покупателей. Определение возраста необходимо для двух задач:

- анализ покупательского поведения и предложение товаров, релевантных возрастной группе;
- контроль соблюдения правил продажи возрастно-ограниченных товаров (например, алкоголя).

**Цель:** построить модель, которая по фотографии человека определяет его приблизительный возраст.

## 2. Входные данные

Набор фотографий людей с указанием возраста (разметка предоставлена).

**Характер данных:**
- изображения в формате, пригодном для обучения нейронных сетей (JPEG/PNG);
- для каждого изображения задан целевой признак — возраст (в годах).

## 3. Этапы выполнения

### 3.1. Исследовательский анализ данных
- Оценить общее количество изображений.
- Проанализировать распределение возраста в выборке (гистограмма, основные статистики).
- Визуализировать примеры изображений для разных возрастных групп.
- Выявить потенциальные проблемы:
  - несбалансированность возрастных групп;
  - низкое качество изображений;
  - наличие артефактов.
- Сформулировать выводы, влияющие на подготовку данных и выбор архитектуры.

### 3.2. Подготовка данных к обучению
- Привести все изображения к единому размеру (например, 224×224 или 128×128 пикселей).
- Нормализовать значения пикселей (приведение к диапазону [0,1] или стандартизация).
- Разделить выборку на обучающую, валидационную и тестовую (например, 70% / 15% / 15%) с сохранением распределения возраста.
- Применить аугментацию для обучающей выборки:
  - повороты;
  - сдвиги;
  - отражения;
  - изменение яркости/контраста.
- Сбалансировать выборку при необходимости (взвешивание классов или oversampling).

### 3.3. Построение и обучение нейронной сети

**Задача:** регрессия (предсказание возраста как непрерывной величины).

**Требования к модели:**
- использовать свёрточные нейронные сети (CNN);
- в качестве основы допустимо использовать предобученные архитектуры (ResNet, EfficientNet, VGG и др.) с дообучением (fine-tuning).

**Архитектуры для рассмотрения:**
- базовая свёрточная сеть (с нуля);
- предобученная сеть (с заморозкой части слоёв);
- предобученная сеть с полной донастройкой.

**Функция потерь:** MAE (средняя абсолютная ошибка) или MSE.

**Метрика качества:** MAE (средняя абсолютная ошибка в годах).

**Процесс обучения:**
- подобрать скорость обучения;
- применить раннюю остановку (early stopping) для предотвращения переобучения;
- использовать регуляризацию (Dropout, L2);
- отслеживать динамику ошибки на обучающей и валидационной выборках.

### 3.4. Оценка качества модели
- Рассчитать MAE на тестовой выборке.
- Визуализировать предсказания vs фактические значения.
- Проанализировать ошибки по возрастным группам (модель систематически ошибается на детях / пожилых и т.д.).
- Сравнить качество разных архитектур.

### 3.5. Выводы
- Указать лучшую модель и её значение MAE.
- Оценить пригодность модели для практического использования.
- Предложить направления улучшения (увеличение выборки, более сложная аугментация, ансамблирование).

## 4. Выходные результаты

- Jupyter Notebook с полным кодом: EDA, подготовка данных, обучение, оценка.
- Лучшая обученная модель (сохранённые веса).
- Значение MAE на тестовой выборке.
- Выводы о качестве модели и возможности её применения.

## 5. Примечания

- Задача решается методами компьютерного зрения (свёрточные нейронные сети).
- Данные представляют собой изображения людей — требуется соблюдение этических норм и обезличивания.
- Основной критерий успеха — низкая средняя абсолютная ошибка предсказания возраста.
